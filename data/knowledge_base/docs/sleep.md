# 睡眠卫生

睡眠和情绪状态关系密切。睡眠不足会降低注意力、记忆力和情绪调节能力，也会放大焦虑、烦躁和负面想法。改善睡眠不一定需要复杂方法，稳定的睡眠习惯往往最重要。

睡眠卫生建议包括：尽量固定起床时间；睡前一小时减少高刺激活动；避免在床上长时间刷手机；下午或晚上减少咖啡、浓茶和能量饮料；白天适当晒太阳和运动；把床主要用于睡眠而不是学习、娱乐和焦虑反刍。

如果躺下二十到三十分钟仍然睡不着，可以先离开床，做低刺激活动，例如听轻音乐、阅读纸质书、做放松练习。等困意回来后再上床，避免让大脑把床和焦虑绑定在一起。

睡前反复想事情时，可以准备一张“明天再处理”清单，把担心的事写下来，并写出第二天最小可执行步骤。这样做不是解决所有问题，而是告诉大脑：这些事已经被暂时存放，不需要在睡前反复提醒。

如果失眠持续数周，或伴随明显情绪低落、焦虑发作、白天功能受损，建议联系专业人员评估。
