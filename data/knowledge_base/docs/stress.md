# 压力管理

压力来自个体对任务要求和自身资源之间差距的感受。大学生常见压力来源包括考试、绩点、就业、人际关系、家庭期待、经济压力和自我比较。压力本身不是坏事，但长期高压会让人疲惫、易怒、效率下降，也可能引发睡眠和身体不适。

管理压力的第一步是区分“可控制”和“暂时不可控制”的部分。可控制的部分可以转化为行动清单，例如复习计划、沟通安排、资料整理和求助对象。不可控制的部分则更适合用接纳、转移注意力和情绪调节来处理。

一个简单方法是使用“三列清单”：我正在担心什么、我能做的一小步是什么、我需要谁的帮助。这个方法能把模糊压力变成具体行动，减少大脑反复空转。

压力很大时，可以先做短时间恢复，比如走路十分钟、洗澡、整理桌面、做三轮深呼吸或暂时离开屏幕。恢复不是偷懒，而是让大脑重新获得调节能力。

如果压力伴随持续失眠、明显食欲变化、强烈无助感或自伤想法，应优先寻求专业帮助。
