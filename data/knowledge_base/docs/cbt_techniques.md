# CBT 常用自助技巧

认知行为疗法常用一个基本思路：情绪并不只由事件本身造成，也和我们对事件的解释有关。例如“考试没考好”可能引发“我彻底失败了”的想法，这种想法会进一步带来焦虑、羞耻和逃避。

一个适合自助练习的方法是“想法记录”。可以写下五项内容：发生了什么、我当时的情绪是什么、我脑中自动冒出的想法是什么、支持这个想法的证据是什么、不支持这个想法的证据是什么。最后尝试写一个更平衡的替代想法。

常见认知偏差包括灾难化、非黑即白、过度概括、读心术和否定积极面。识别偏差不是批评自己，而是给自己一个机会，用更完整的信息看待问题。

行为激活也是常见技巧。当人情绪低落时，往往会减少活动，而活动减少又会让情绪更低。可以从非常小的活动开始，例如整理床铺、下楼买水、给朋友发一条消息、走五分钟。重点不是立刻开心，而是恢复行动感。

这些技巧适合轻中度压力和情绪困扰的自我调节。如果情绪问题持续加重，或出现危机信号，应联系专业人员。
